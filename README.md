
# Expenzo - Expenses Management System 💰

## 📌 Project Description
Expenzo is a web-based expense management system that helps users track their income and expenses efficiently. It provides budget management and reporting features for better financial planning.

## 🚀 Features
- User Authentication (Login & Registration)
- Add and Manage Expenses & Income
- Budget Setting and Notifications
- Reports Generation
- User Profile Management

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## ⚙️ How to Run the Project
1. Install XAMPP / WAMP
2. Import the database.sql file into phpMyAdmin
3. Place the project folder in `htdocs`
4. Start Apache & MySQL
5. Open `localhost/project-folder-name`
